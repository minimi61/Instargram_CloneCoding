## ****🤷‍♂️ 프로젝트 목적****
이 프로젝트는 항해99의 클론 프로젝트 과제로<br>
상업목적이 아닌 개인의 기술역량 강화를 초점으로 만들어졌으며 <br>
나아가 협업을 배우고 백엔드 - 프론트엔드 커뮤니케이션을 익히려는 목적으로 만들어졌습니다.<br><br>
